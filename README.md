# books-api

## How to run this project?
* Clone the repository
`git clone https://github.com/AjasEusebio/books-api.git`
* Install dependencies
`npm install`
* Execute project 
`npm run dev`
* Move to `/src/requests` folder and test endpoints with some API testing tool.
